# Push Guide

```bash
# Initialize and push
git init
git add .
git commit -m "Initial commit: MERN e-commerce with Desert Modern UI"
git branch -M main
git remote add origin https://github.com/WassimBenMechlia/e-commerce-web-proj.git
git push -u origin main

# For subsequent updates
git add .
git commit -m "descriptive message"
git push origin main
```
